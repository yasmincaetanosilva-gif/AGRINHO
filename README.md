# 🌱 SustentAgro

O **SustentAgro** é uma plataforma digital e educativa desenvolvida para auxiliar produtores rurais e entusiastas do setor agrícola na tomada de decisões sustentáveis. Utilizando dados simulados de clima e características do solo, o sistema oferece cálculos de irrigação precisa e recomendações inteligentes para otimizar o uso da água e incentivar práticas agrícolas responsáveis.

Este projeto foi orgulhosamente desenvolvido para o **Programa Agrinho**.

---

## 🚀 Funcionalidades

* **🌦️ Simulador de Clima Dinâmico:** Altere o cenário climático da aplicação (Sol, Chuva ou Seca) com um clique e veja a interface e as diretrizes se adaptarem visualmente e contextualmente.
* **💧 Calculadora de Irrigação Inteligente:** Insira o tipo de solo (Arenoso, Argiloso ou Humoso) e o tamanho da área plantada (em hectares) para receber instantaneamente a quantidade de água necessária por dia e o nível de sustentabilidade do manejo.
* **📊 Dashboard Ambiental Animado:** Gráficos de barras interativos que ilustram indicadores ecológicos como economia de água, sustentabilidade geral, uso de energia limpa e eficiência agrícola.
* **🧠 Recomendações Personalizadas:** Um sistema lógico que cruza o estado do clima atual com o tipo de solo selecionado para gerar alertas e dicas cruciais de manejo hídrico sustentável.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web fundamentais (Vanilla Architecture), garantindo leveza, acessibilidade e performance:

* **HTML5:** Estruturação semântica de todas as seções e simuladores do site.
* **CSS3:** Estilização moderna, design responsivo, uso de efeitos de desfoque (`backdrop-filter`) e animações fluidas (`@keyframes`) nos componentes do dashboard.
* **JavaScript (ES6):** Manipulação dinâmica do DOM, lógica de condições para tomada de decisões e cálculos matemáticos baseados nas variáveis inseridas pelo usuário.
